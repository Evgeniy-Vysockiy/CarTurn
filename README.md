Мобильное приложение, разработанное с использованием мульплатформенной среды разработки Unity 3d.
Суть приложения: пользователь нажатием на дисплей мобильного телефона должен "запустить" автомбиль в движущийся поток автомобилей
Проект был разработан в качестве учебного, для оттачивания навыков написания кода и улучшения понимания ООП
