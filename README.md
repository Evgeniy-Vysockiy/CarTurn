#GameDev

Мобильное игровое приложение, разработанное с использованием мультиплатформенной среды разработки Unity 3d. 
Геймплей игры: пользователь нажатием на дисплей мобильного телефона должен "запустить" автомобиль в движущийся поток автомобилей Проект был разработан в качестве учебного, для оттачивания навыков написания кода и получения опыта в оптимизации приложений.
