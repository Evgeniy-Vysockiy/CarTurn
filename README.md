Мобильное приложение, разработанное с использованием мультиплатформенной среды разработки Unity 3d.
Суть приложения: пользователь нажатием на дисплей мобильного телефона должен "запустить" автомобиль в движущийся поток автомобилей
Проект был разработан в качестве учебного, для оттачивания навыков написания кода и улучшения понимания ООП
